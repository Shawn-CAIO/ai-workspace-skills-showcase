# My Content Digest

가져온 콘텐츠(트윗, YouTube 등)를 Quiz-First 방식으로 소화하는 스킬.
요약을 먼저 보여주지 않고, 퀴즈부터 출제하여 학습 효과를 극대화한다.

## 핵심 원칙

**요약을 먼저 보여주지 않는다.**

```
❌ 일반적 순서: 요약 읽기 → 퀴즈
✅ Quiz-First: 퀴즈 먼저 → 틀린 부분 확인 → 선택적으로 읽기
```

- Pretesting Effect: 학습 전 테스트가 기억력을 9-12% 향상
- Information Gap Theory: 틀린 문제가 호기심을 만들고, 호기심이 기억을 강화

## Pre-Quiz (콘텐츠 보기 전 퀴즈)

### 출제 절차

1. 콘텐츠에서 핵심 주제 3개를 추출한다
2. AskUserQuestion으로 3문제를 출제한다:
   - 핵심 개념 관련 1문제
   - 저자 핵심 주장 관련 1문제
   - 주요 원칙 관련 1문제
3. 각 문제는 4지선다 (label + description)

### 출제 원칙

- ✅ 핵심 메시지, 주요 원칙, 저자 핵심 주장
- ❌ 날짜, 통계 수치, 지엽적 세부사항

### 피드백

각 문제에 대해 즉시 정답/오답 피드백을 제공한다.
오답인 경우: "이 부분을 콘텐츠에서 확인해보세요" 안내를 포함한다.

## 선택적 콘텐츠 제공

Pre-Quiz 결과에 따라 AskUserQuestion으로 사용자가 선택한다:

- **"틀린 문제 관련 섹션만"** — 오답 부분의 답을 콘텐츠에서 찾기
- **"핵심 인사이트 3개"** — 가장 중요한 포인트만
- **"전체 요약 + 인사이트"** — 종합적 분석
- **"바로 본 퀴즈로"** — 요약 없이 9문제 도전

## 본 퀴즈 (9문제)

3단계 × 3문제 = 총 9문제. 각 단계별로 AskUserQuestion을 사용한다.

### 기본 (3문제)

- 핵심 메시지 이해
- 주요 개념 파악
- 저자의 핵심 주장

### 중급 (3문제)

- 개념 간 관계 연결
- 근거와 주장의 연결
- 맥락 속 의미 파악

### 심화 (3문제)

- 실제 상황에 적용
- 핵심 원리의 확장
- 비판적 분석

### 출제 원칙

- ✅ 핵심 주제만 출제
- ❌ 지엽적 세부사항, 날짜, 통계 수치 금지

각 단계 완료 후 즉시 정답/해설을 제공한다.

## 후속 선택

본 퀴즈 완료 후 AskUserQuestion으로 후속 행동을 선택받는다:

- **"다른 문제로 재퀴즈"** — 새로운 9문제
- **"관련 콘텐츠 추천"** — 주제 관련 추가 학습 자료
- **"종료"** — 학습 마무리

Claude Code Max를 본격적으로 쓰기 시작하는 분들에게 도움을 주고자, 제가 한 달째 실무와 개인 일정 관리에 쓰고 있는 개인 ai-workspace 셋업을 공유합니다.

AI 교육·컨설팅을 하면서 반복되는 업무를 하나씩 자동화해왔습니다. 현재 9개 커스텀 스킬이 매일 돌아가고 있고, 이 강의에서는 그 셋업 전체를 열어봅니다.

다루는 내용

  • 매일 아침 Slack + Webex + 소마 마이페이지 + Gmail + Calendar를 한 곳으로 모으는 context sync
  • 캘린더 앞뒤 일정의 location을 읽어 이동경로를 자동 등록하는 route planner
  • Playwright 기반 외부 Slack 워크스페이스 스크래퍼
  • 세션 종료 시 작업 정리·학습 기록을 남기는 session wrap
  • 커스텀 스킬의 구조와 만드는 법 — SKILL.md, install-script, 맥 간 동기화
  • 메모리 시스템 — 대화가 끝나도 맥락이 유지되는 구조
  • 이 멘토특강 기획 자체가 만들어진 과정 — 800건 크롤링 → 분석 → Notion 발행

이런 분에게

  • Claude Code Max를 곧 쓸 예정인데 뭐부터 세팅할지 참고가 필요한 분
  • Cursor, ChatGPT 외에 다른 워크플로우를 구경해보고 싶은 분
  • 반복 업무를 줄이고 싶은데 어디서부터 시작할지 감이 안 잡히는 분
